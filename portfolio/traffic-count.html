Tahoe Traffic Count Interactive

An interactive web application visualizing decades of historical traffic count data collected at monitoring stations throughout the Lake Tahoe region.

View Live
// live preview // about this project

TRPA maintains a network of traffic monitoring stations around the Lake Tahoe basin, collecting annual vehicle count data to inform transportation planning, corridor studies, and environmental threshold compliance.

This application makes that data interactive — users can click any station on the map to view a time-series chart of annual counts, filter by corridor or station type, and compare trends across the region. The goal was to move a dataset that previously lived in spreadsheets into a spatial context where patterns become visible.
