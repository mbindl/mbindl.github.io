Tahoe Planning Tools

A suite of web maps, dashboards, and infographics supporting environmental planning and public communication in the Lake Tahoe basin.

View Live
// live preview // about this project

The Tahoe Planning Tools collection is an ongoing suite of interactive web maps and data dashboards built to support environmental planning, regulatory compliance, and public outreach at the Tahoe Regional Planning Agency.

Each tool is designed to make complex geospatial and environmental data accessible — to agency staff, local governments, and the public alike. The suite spans water quality monitoring, land capability, permit tracking, and regional condition indicators.
